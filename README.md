SQL Case Study: Tech Expansion Strategy (Magist x Eniac)
**🎯 The Mission**
Evaluate if the Brazilian marketplace Magist is a viable partner for Eniac (High-end Tech) by analyzing 100k+ orders for logistics efficiency and market demand.

**🚀 Key Findings**
The Growth: Computer Accessory sales exploded by 75% YoY.
The Gap: A total "Market Vacuum" exists; only one seller earns >€8k/month in Tech.
The Risk: Logistics "Warehouse Speed" is slowing down as volume scales.
The Error: Qualitative data shows "Wrong Product" errors, indicating a need for SKU scanning.

**🛠️ Tech Stack**
SQL (MySQL): Time-series analysis, CASE logic, and multi-table joins.
Tableau: Sheets for fulfillment bottlenecks.
Canva: Presentation.

**📁 How to use this repo**
/Queries: Find the logic for the "Tech vs. General" market share shift.
/Results: See the final recommendation slide deck.
